This snake Game is build in c Language 
using concept of structures in them

Its is to be played on linux Terminal

>>Controls:..
w-moves UP
a-moves left
s-moves down
d-moves right 
q-quit the game


To run
1.Download the file attached
2.open terminal on the location where file is kept
3.write "g++ snakeg.cpp"
4.Then enter cmnd "./a.out"
5.enjoy the snake game
